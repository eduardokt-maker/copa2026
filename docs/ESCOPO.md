# Escopo Inicial

## Nome do projeto

copa2026

## Descricao

Projeto novo, independente e separado do sistema atual.

Primeira entrega: tela de apresentacao com as 48 selecoes participantes da Copa 2026, pais de origem, bandeira, filtro por confederacao e busca.

## Fontes iniciais

- Lista de finalistas: pagina publica "2026 FIFA World Cup".
- Paises-sede: Canada, Mexico e Estados Unidos.
- Bandeiras: imagens publicas via FlagCDN.

## Decisoes pendentes

- Tipo de produto: site, app, painel, sistema administrativo, API ou outro.
- Publico-alvo.
- Fonte dos dados.
- Tecnologia.
- Hospedagem.
